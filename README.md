## Quick Start

- Clone the repo: `git clone https://github.com/wikiakai/Cipta-Integrasi-Nusantara.git`
- open terminal
- Install dependencies: `npm install` or `yarn`
- Start the server: `npm run dev` or `yarn run dev`
- Views are on: `localhost:5173`

## Tech Stack
- React : 18.2.0
- Material UI : 5.11.0
- Axios : 1.4.0
